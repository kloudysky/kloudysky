Smartphone Car

Smartphone Car is a Vanilla JS driving simulator that transforms your phone into a gyroscope-based controller. It creates a unique connection between your phone and the computer using Socket.IO and listens for movement via the DeviceMotion API

Smartphone Car는 당신의 폰을 자이로스콥 컨트롤러로 바꿔주는 바닐라 JS 운전 시뮬레이터입니다. Socket.IO를 통해 폰과 컴퓨터 사이를 특수하게 연결해주고 DeviceMotion API를 이용해 움직임을 감지합니다.

JavaScript Three.JS Socket.IO Express

Reddit Lite

There's a lot more to this app that goes on under the hood. On the frontend is a lite clone of Reddit. Behind the scenes there is a GraphQL server using TypeScript with Type-GraphQL as the GraphQL schema. Query/mutation requests, caching and SSR with URQL. Cursor pagination and more. Checking out the repo!

이 앱에는 더 많은것들이 담겨있습니다. 겉보기엔 Reddit의 복제품이지만 알고보면 GraphQL schema로 Type-GraphOL을 TypeScript을 사용한 GraphQL 서버가 있습니다. 쿼리/뮤테이션 리퀘스트, 캐싱 그리고 URQL. 커서 페이징, 그 외에도 다양한 기능이 숨겨져 있습니다. GitHub을 확인해보세요!

TypeScript TypeORM GraphQL URQL Redis

Chakra-UI Next.JS React

Blockup

Blockup is communication MERN app that enforces 2FA after sign up and allows users to chat, talk, and video call other users or groups. I Managed a team of 3 engineers to meet all milestone deadlines while ensuring git workflow best practices.

Blockup은 커뮤니케이션 MERN 앱으로 가입 후 2FA를 강화하고 사용자들이 문자, 음성, 그리고 비디오로 대화할 수 있게합니다. 저는 세명의 엔지니어들이 최선을 다하면서도 주어진 기간 내에 목적을 달성하도록 이끌었습니다.
